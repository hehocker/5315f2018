What is Docker

Docker is a computer program that performs containerization. According to Docker.com, a container is a standard unit of software that contains all the required code and associated software (dependencies). By using containers, programmers can make sure that software always runs the same way and delivers consistent and reproducible results. Therefore, it is easy to share containers with co-workers or programming partners. Sources:https://www.docker.com/,https://ropenscilabs.github.io/r-docker-tutorial/01-what-and-why.html,https://askubuntu.com/questions/361741/what-are-dependencies

What is Nginx

Nginx is opensource software that can be used as a webserver. Nginx can also be used as a load balancer (to distribute the traffic to the site) or as a proxy server for email. The founder of Ngninx wanted to make the fastest web server. According to Kinsta, Nginx works with one master process controlling multiple worker processes. The worker processes do the actual work while the master process maintains the workers. Sources:https://www.nginx.com/resources/glossary/nginx/, https://www.netguru.co/codestories/nginx-tutorial-basics-concepts, https://kinsta.com/knowledgebase/what-is-nginx/)

What is VM (virtual machine)

A virtual machine is a computer file that works like a computer. Another name for the computer file in an image. Users work on a virtual machine exactly the same way they would on a physical computer. A phycial computer can run multiple virtual machines (with different operating systems and other features) at the same time. Sources: http://www.serial-server.net/virtual-machine/, https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/ .

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a cloud computing platform. They provide Infrastructre as a Service (IaaS). Users can choose the operating system or configuration of their virtual machine (or droplet as Digital Ocean calls them). Cloud computing service providers may also sell software as a service (SaaS) like Microsoft Office 365 or provide a platform as a service (PaaS) like Windows Azure. Sources: https://www.digitalocean.com/community/questions/is-digitalocean-iaas-or-paas-platform, https://stackoverflow.com/questions/16820336/what-is-saas-paas-and-iaas-with-examples, https://www.digitalocean.com/.
